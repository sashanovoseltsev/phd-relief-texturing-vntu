# Document Formatting Rules

## University Text Assignments

- Default storage location: Google Drive folder `PhD/Tasks`, unless the user specifies another folder.
- Default font: Times New Roman.
- Default font size: 14 pt.
- Do not add an extra blank line between paragraphs.
- Add extra blank line between sections
- Format section and paragraph titles in bold.
- Format mathematical formulas as native Google Docs equations through `Insert -> Equation` when the editing surface supports it.
- Do not silently replace native equations with plain text; if only API text editing is available, keep formulas readable as linear text and state the limitation.
- Use Ukrainian for Ukrainian university assignments unless the user asks for another language.
- Keep structure clear: title, short introductory context, numbered task sections, conclusions or expected results where appropriate.
- For dissertation-related assignments, connect the content to the active PhD topic on relief texturing, parallax mapping, POM, relief mapping, cone step mapping, and Unity Desktop shader experiments.
