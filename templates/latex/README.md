# LaTeX Templates

## Ukrainian University Assignment

Template:

`templates/latex/ukrainian-university-assignment-template.tex`

Use it for Ukrainian university text assignments that need:

- A4 page format;
- Times New Roman, 14 pt;
- 1.5 line spacing;
- one line of vertical space between major sections;
- bold section titles;
- Ukrainian text with English technical terms in parentheses;
- displayed equations through `amsmath`;
- centered `Список використаних джерел`;
- numbered APA-like references.

To create a new work, copy the template into the target task folder, rename it, edit the title/content, and compile:

```bash
tectonic your-file.tex
```
